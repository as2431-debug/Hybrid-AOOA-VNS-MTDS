# Hybrid AOOA-VNS for MIDS

## Description
This project presents an improved version of the AOOA algorithm using 
VNS (Variable Neighborhood Search) to solve the Minimum Independent Dominating Set (MIDS) problem.

## Contribution
The main improvement is replacing the worker life cycle with VNS to enhance solution quality.

## Files
- main.py : main implementation
- dataset : graph instances used in experiments

## How to run
Run the code using Python:
python main.py
